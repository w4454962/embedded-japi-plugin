# 编辑框 class.edit

继承自 [面板类](Script/界面/面板)

# 属性

* [构造器](Script/界面/构造器)

* 以下属性只有在构造时填入 在此之后只读不写

名字|类型|说明
--|--|--
font_size|number|字体大小 从1~25

# 方法


### set_text

设置文本

#### 参数
名字|类型|说明
--|--|--
text|string| 主动设置一次文本


```lua
    edit:set_text(text)
```

### get_text

获取文本

#### 返回
索引|类型|说明 
1|string| 返回编辑框的文本

```lua
    local text = edit:get_text()
```

### set_focus

设置焦点

#### 参数
名字|类型|说明
--|--|--
is_enable|boolean| 开启后键盘输入在里面


```lua
    edit:set_focus(is_enable)
```
